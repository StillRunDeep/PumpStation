# 00 计算定义总纲

> 本文件是雨水泵站智能设计系统的统一参考文档，定义所有 AG 共用的本体对象、管道命名约定、共享常量和数据流关系。各 AG 文档中不重复定义这些内容，只引用本文件。

---

## 一、系统功能分层

系统按功能分为四大层次：

```
┌─ 用户交互（AG0-x）────────────────────────────────────────────┐
│  AG0-0  用户对话        → 收集输入参数、验证、Q_single 计算    │
│  AG0-1  连接关系配置器  → ConnectionTopology（管道拓扑对象）   │
└───────────────────────────────────────────────────────────────┘

┌─ 水池计算（AG1-x）────────────────────────────────────────────┐
│  AG1-1  污水池计算      → V_min, h_wetWell 或 S_min, 三水位   │
└───────────────────────────────────────────────────────────────┘

┌─ 水泵与管道（AG1-x）──────────────────────────────────────────┐
│  AG1-2  水泵选型        → H, P_motor                          │
│  AG1-3  管道尺寸计算    → DN_pumpIn, DN_pumpOut, DN_mainOutlet │
│  AG2-3  小型排水泵      → SUP 参数（独立分支，不进主流程）     │
└───────────────────────────────────────────────────────────────┘

┌─ 图形表达（AG3-x）────────────────────────────────────────────┐
│  AG3-1  设备流线二维示意 → 维护间尺寸 L/W + SVG 图纸          │
└───────────────────────────────────────────────────────────────┘
```

> AG4-2/AG4-3（建筑平面布局）单独维护，不在本总纲范围内。

## 二、数据流总览

```
AG0-0 用户对话
  │  Q（总流量）, N（泵数）, DischargeLevel, L（管长）
  │  + 模式选择：已知 S 或 已知 h_wetWell
  │  Q_single = Q / N（AG0-0 直接计算输出）
  │
  ├──► AG1-1 污水池计算
  │       输入：Q_single（AG0-0）, Z, S 或 h_wetWell
  │       输出：V_min, h_wetWell, S_min, stopLevel, startLevel, alarmLevel
  │             ↓
  ├──► AG1-2 水泵选型
  │       输入：Q_single（AG0-0）, stopLevel（AG1-1）, DischargeLevel, L
  │       输出：H, P_motor
  │             ↓
  ├──► AG1-3 管道尺寸计算
  │       输入：Q_single（AG0-0）, Q, N
  │       输出：DN_pumpIn, DN_pumpOut, DN_mainOutlet
  │
  ├──► AG0-1 连接关系配置器（用户交互）
  │       输入：N（AG0-0）, DN_*（AG1-3）, P_motor（AG1-2）
  │       输出：ConnectionTopology（节点 + 边）
  │             ↓
  ├──► AG3-1 设备流线二维示意
  │       输入：ConnectionTopology（AG0-1）, P_motor（AG1-2）,
  │             DN_*（AG1-3）, WetWell 水位（AG1-1）
  │       输出：维护间净长 L_room, 净宽 W_room, SVG 图纸
  │
  └──► AG2-3 小型排水泵（独立分支）
          输入：合同固定参数（Clause 41.57）
          输出：SUP 规格参数（独立展示，不进主流程）
```

> Q_single = Q / N 在 AG0-0 输出阶段直接计算，供 AG1-1、AG1-2、AG1-3 共同使用，避免循环依赖。

## 三、本体对象定义

### 3.1 Pump（水泵）

| 属性 | 类型 | 单位（SI） | 说明 |
|------|------|-----------|------|
| `flowRate` | Float | m³/h | 单台水泵设计流量（= Q_single） |
| `head` | Float | m | 系统设计扬程 |
| `motorPower` | Float | kW | 配套电机功率 |
| `efficiency` | Float | 无量纲 | 运行效率（0～1），选型前取预估值 0.70 |
| `isStandby` | Bool | — | true = 备用泵，不计入工作台数 N |

---

### 3.2 WetWell（污水池）

> 英文：`WetWell`，中文：污水池（原名 CollectionPool）

| 属性 | 类型 | 单位（SI） | 说明 |
|------|------|-----------|------|
| `effectiveVolume` | Float | m³ | 最小有效调节容积（= V_min） |
| `depth` | Float | m | 湿井总有效深度（= h_wetWell） |
| `footprintArea` | Float | m² | 平面占地面积（= S） |
| `activeDepth` | Float | m | 有效调节水深（= h_active） |
| `stopLevel` | Float | m | 停泵水位，距池底 |
| `startLevel` | Float | m | 首台启泵水位，距池底 |
| `alarmLevel` | Float | m | 报警水位，距池底 |
| `bottomSlope` | Float | 无量纲 | 底部坡度，≥ 1:10 |

---

### 3.3 Motor（电机）

| 属性 | 类型 | 单位（SI） | 说明 |
|------|------|-----------|------|
| `ratedPower` | Float | kW | 电机额定功率（= P_motor） |
| `maxStartsPerHour` | Int | 次/h | 每小时最大允许启动次数，默认 6 |

---

### 3.4 管道对象命名约定

> 注：系统采用潜污泵（Submersible Pump），无吸入管（PumpInlet）。

系统中有 3 类管道，命名明确区分：

| 对象名 | 中文名 | 流向 | 管径参数 | 计算模块 |
|--------|--------|------|----------|----------|
| `WetWellInlet` | 湿井进水管 | 外部雨水 → 湿井 | 由外部条件给定 | 不在本系统计算 |
| `PumpOutlet` | 水泵压水管 | 水泵出口 → 汇总干管 | `DN_pumpOut` | AG1-3 |
| `MainOutlet` | 总出水管 | 汇总干管 → 最终排放点 | `DN_mainOutlet` | AG1-3 |

---

### 3.5 阀件对象

| 对象名 | 中文名 | 典型位置 |
|--------|--------|----------|
| `CheckValve` | 止回阀 | 每台泵 PumpOutlet 上 |
| `GateValve` | 闸阀 | CheckValve 外侧 |
| `ButterflyValve` | 蝶阀 | 可替代 GateValve（用户可配置） |

## 九、水泵与管道设计参数（手册第14章）

### 9.1 效率要求

| 设备 | 参数 | 要求值 | 依据 |
|-----|------|-------|------|
| 水泵 | η_hyd | ≥ 0.75 | 手册第14.6节 |
| 电机 | η_mot | ≥ 0.85 | 手册第14.6节 |

**轴功率公式**（手册第14.6.2节）：
```
P_shaft = ρ × g × Q × H / (η_hyd × η_mot)    （kW）
```
其中：ρ = 1000 kg/m³，g = 9.81 m/s²

### 9.2 电机安全系数

| 轴功率范围 | 安全系数 K |
|-----------|-----------|
| P_shaft < 15 kW | 1.25 |
| 15 ≤ P_shaft < 55 kW | 1.15 |
| P_shaft ≥ 55 kW | 1.10 |

**电机功率**：
```
P_motor = P_shaft × K    （kW）
```

### 9.3 管道设计流速

> 注：系统采用潜污泵（Submersible Pump），无吸入管设计。

| 管道类型 | 设计流速 | 依据 |
|---------|---------|------|
| 出水管（压水管） | 1.5-2.4 m/s | 经济流速 |
| 总出水管 | 1.5-2.2 m/s | 合流干管 |

### 9.4 曼宁公式沿程损失

> 依据：手册第8.3节

```
H_f = 10.29 × n² × L × Q² / D^5.33    （m）
```

**曼宁粗糙系数**：

| 管材 | n |
|-----|---|
| 混凝土 | 0.013-0.015 |
| 钢管 | 0.012-0.014 |
| 铸铁管 | 0.012-0.015 |

## 十、汽蚀余量（NPSH）校验

### 10.1 必需汽蚀余量

| 参数 | 符号 | 说明 |
|------|------|------|
| 大泵 NPSH_r | 3-5 m | 大型轴流泵典型值 |
| 小泵 NPSH_r | 2-3 m | 中型离心泵典型值 |

### 10.2 有效汽蚀余量

```
NPSH_a = (P_atm - P_v) / (ρg) + H_s - H_suction_loss    （m）
```

**要求**：
```
NPSH_a ≥ NPSH_r + 0.5 m    （安全余量）
```

## 五、单位约定

### 5.1 架构原则

```
用户输入（可选单位）
    ↓  AG0-0：统一转换为 SI
计算层（AG1-x / AG2-x）← 始终使用 SI 单位
    ↓  显示层：转换回用户选择的单位
结果展示
```

计算模块只使用 SI，不做单位判断。

### 5.2 换算系数表

| 物理量 | SI 单位 | 换算单位 | 换算关系 |
|--------|---------|----------|----------|
| 流量 | m³/h | L/s | 1 m³/h = 1/3.6 L/s |
| 流量 | m³/h | GPM | 1 m³/h ≈ 4.403 GPM |
| 流量 | m³/h | MGD | 1 m³/h ≈ 0.00634 MGD |
| 扬程/深度 | m | ft | 1 m = 3.2808 ft |
| 功率 | kW | HP | 1 kW = 1.341 HP |
| 管径 | mm | inch | 1 inch = 25.4 mm |
| 面积 | m² | ft² | 1 m² = 10.764 ft² |
