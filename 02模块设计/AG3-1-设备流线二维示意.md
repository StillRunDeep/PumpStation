# AG3-1 设备流线二维示意

> 功能分类：**图形表达**
> 本体对象定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG3-1 根据 AG0-1 输出的连接拓扑和 AG2-1 计算的空间尺寸规则，将所有设备按实际比例摆入维护间，并生成两个视图：

- **平面图（俯视）**：设备水平布局、管件（弯头/止回阀/闸阀/流量计）按实际比例绘制
- **A-A 剖面图**：竖向空间关系，三水位标注，切割线可拖动

---

## 二、在系统中的位置

```
AG0-1（连接关系配置器）  → ConnectionTopology
AG1-2（水泵选型）        → P_motor, catalogPump（含 dimensions_mm）
AG1-3（管道尺寸计算）    → DN_pumpIn, DN_pumpOut, DN_mainOutlet
AG1-1（污水池计算）      → h_wetWell, stopLevel, startLevel, alarmLevel
AG2-1（维护间尺寸）      → L, W, hasCatalogDims, c_wall_m, L_elbow_m, DN_branch, DN_main
AG0-0（用户对话）        → N
  ▼
AG3-1（设备流线二维示意）
  │  输出：SVG 平面图 + A-A 剖面图
  ▼
网页展示区域
```

## 五、绘制内容

### 5.1 坐标系

| 视图 | X 轴 | Y 轴 | 说明 |
|------|------|------|------|
| 平面图 | 向右（泵排列方向）| 向下（通道方向）| 俯视 |
| A-A 剖面图 | 室内净宽 W | 向上（高程）| 竖向 |

两视图并列于同一 SVG 画布；剖面切线在平面图中以虚线及"A"符号标注。

### 5.2 平面图绘制对象

> 注：潜污泵布局特点 — 泵在集水坑（wet_well）内，完全浸没；维护间内只有管道和阀件。

| 对象 | 绘制方式 | 位置依据 |
|------|----------|----------|
| 维护间外轮廓 | 矩形，L × W | 全局边界 |
| 集水坑区域 | 矩形，面积 = S（或 S_min）| 维护间进水侧（room_y2 下方）|
| 水泵机组 × N_total | 矩形块，`w_pump × d_pump` | **在集水坑区域内**，沿 X 方向等间距排列 |
| 压水管（竖向穿楼板）| 虚线圆圈（楼板穿孔）+ 竖向虚线 | 从集水坑向上穿 room_y2 进入维护间 |
| 分支管路（维护间内）| 单线，水平方向（Y 轴方向）| 从穿楼板孔向总管方向延伸 |
| 止回阀、闸阀 | 标准符号，竖向布置（horiz=false）| 在分支管路上 |
| MainOutlet 干管 | 横向粗线（2.5px），水平 | 紧靠维护间背墙（hdr_y） |
| 旁通管道（若有）| 灰蓝色虚线 | 从总管引出绕集水坑边缘回进水 |
| 控制柜 | 矩形块 | 维护间端部角落 |
| DN 标注 | 单线旁标注 DNxxx，空间不足用引线 | 每段管道旁 |
| Q 标注 | 泵上方标注 Q=xxx m³/h | 来自 extraInfo.Q_single |
| 剖面切线 A-A | BS EN ISO 128 标准符号 + 可拖动 | 沿 Y 轴拖动 |

**管件绘制细节（平面图）：**

潜污泵无吸入管，压水管从集水坑向上穿楼板进入维护间后，水平延伸连接总管：

1. 楼板穿孔（圆形表示，在 room_y2 位置）
2. 压水管竖向段（虚线，从集水坑内泵出口向上到穿孔）
3. 90° 弯头（弧形，1.5×DN）
4. 分支水平管段（沿 Y 方向，止回阀→直管段→闸阀）
5. 止回阀（菱形符号，竖向，GB/T 12221）
6. 闸阀（矩形符号，竖向，GB/T 12221）
7. 连接到总管（T 型节点）
8. 总管（横向粗线）
9. 总管闸阀 + 电磁流量计

**单线管道绘制：**
- 管道：单线表示（stroke-width: 1.5px），总管 2.5px
- 每段标注 DNxxx（9px，颜色 #555）
- 空间不足（<40px）用 `_leader` 引线

**旁通管道：**
- 检测拓扑中 bypass 设备链路（junction → bypass_gate_valve → bypass_check_valve → source）
- 若存在，画灰蓝色虚线从总管绕集水坑边缘回进水口

### 5.3 A-A 剖面图绘制对象

泵房净高估算：

```
room_H = max(3.0, h_wetWell × 0.2 + 1.0)    （m）
```

| 要素 | 表达方式 | 说明 |
|------|----------|------|
| 室外地坪线（±0.00）| 粗实线 | 地上/地下分界 |
| 泵房围护结构 | 矩形，W × room_H | 地上部分 |
| 污水池池壁 | 矩形，W × h_wetWell | 地下部分 |
| 水面示意 | 蓝色填充 | 从池底到报警水位 |
| 停泵水位 stopLevel | 绿色水平虚线 + 标注 | **左侧**标注 |
| 启泵水位 startLevel | 橙色水平虚线 + 标注 | **右侧**标注 |
| 报警水位 alarmLevel | 红色水平虚线 + 标注 | **左侧**标注 |
| 水泵机组（剖面）| 矩形 | 集水坑内（完全浸没）|
| 压水管（PumpOutlet）| 竖向线段向上 | 从泵出口引出穿楼板 |
| 竖向尺寸标注 | 双向箭头 + 文字 | room_H, h_wetWell |
| 水平尺寸标注 | 双向箭头 + 文字 | W_room |

**水位标注分组（避免文字重叠）：**
- **左侧**（sec_x1 - 6）：停泵水位（anchor='end'）、报警水位
- **右侧**（sec_x2 + 6）：启泵水位

**A-A 切割线可拖动：**

切割线 `<line id="aa-drag-line">` 初始位于泵排中心（Y = pump_row_y），约束在 `room_oy ~ room_y2` 范围内拖动。

拖动时记录 `aaDragY`（module 变量），同步更新 `<line>` 位置 + 重绘剖面内容。

剖面内容按切割线位置切换（沿 W 方向分为三段）：

| 切割线位置（相对 W）| 剖面显示 |
|---------------------|----------|
| 0 ~ 25%（靠集水坑墙）| 泵出口竖管椭圆截面 + 弯头 + 大小头，含 DN 标注 |
| 25% ~ 75%（中间管道区）| 支路管道圆截面 + 止回阀/闸阀符号，含 DN 标注 |
| 75% ~ 100%（靠远侧墙）| 总管圆截面 + 流量计（含 5D/2D 上游/下游标注）|

每段管道圆截面半径 = DN/1000 × ss（含 DN 标注文字）。

### 5.4 剖面符号（BS EN ISO 128）

英国标准剖面切割线符号：

```
    A ←———中心虚线———→ A
    │                   │
    ▼（箭头）           ▼（箭头）
```

- 两端：L 形粗实线（3px）+ 等边三角形箭头
- 中间：细虚线（1px）
- 字母"A"在箭头旁（垂直于切割线方向）
- 可拖动：切割线在平面图内沿 Y 轴拖动，剖面图实时更新

### 5.5 悬停交互

| 元素 | Hover 显示 | 实现方式 |
|------|------------|----------|
| 泵 | 卡片：泵编号、Q(m³/h)、H(m)、P_motor(kW)、型号 | SVG `<g data-*>` + mouseenter/mouseleave 事件 + 绝对定位 HTML div |
| 阀门 | 卡片：阀件名称、DN、面对面长度(mm)、执行标准 | 同上 |

**实现方式：**
- SVG 内嵌 `<title>` 用于简单 tooltip
- 复杂卡片用绝对定位 HTML div（id="pump-tooltip"/"valve-tooltip"），在 index.html 中预定义样式
- 泵 hover 卡片数据来自 `extraInfo`（Q_single, H_design, P_motor, catalogPump）

### 5.6 绘图风格

| 项目 | 规格 |

| 房间填充 | fill="none" 或极浅 #f8fbff |

| 闸阀 | 深红 #922b21 |

### 5.7 剖面组拖动

剖面图所有元素包裹在 `<g id="section-panel" transform="translate(dx,dy)">` 内，`mousedown/mousemove/mouseup` 事件拖动该 transform（用 `stopPropagation` 避免触发整体 SVG 平移）。

reset 按钮回调时重置 dx=dy=0。

### 5.8 最小缩放保护

SVG zoom/pan 限制最小缩放比例：

```
minScale: 1.0    // 初始充满屏幕，不得缩小
maxScale: 6      // 防止过大导致看不到全貌
```

调用 `initSvgZoomPan(el, VW, VH, btnIds, { minScale: 1.0, maxScale: 6, onReset })`。

### 5.9 缩放比例

```
scale_plan = min(
  plan_area_width  / L_pool,
  plan_area_height / (W_room + D_pool)
)    （px/m）

scale_section = min(
  section_area_width  / (W_room + 1.5),
  section_area_height / (room_H + h_wetWell + 0.5)
)    （px/m）
```

## 六、引用规则

| 规则 ID | 简述 |
|---------|------|
| R-LA-01 | 相邻水泵净距 ≥ 1.0 m（大型泵 ≥ 1.2 m）|
| R-LA-02 | 泵距侧墙 ≥ 0.8 m（大型泵 ≥ 1.0 m）|
| R-LA-03 | 大型泵（P_motor > 55 kW）加大间距 |
| R-LA-04 | 每台泵独立吸水管（PumpInlet）|
| R-LA-08 | 就地控制柜布置在角落 |
| R-CM-03 | 停泵水位高于水泵进口 ≥ 0.5 m（剖面图可视验证）|
| R-CM-04 | 水位顺序：stopLevel < startLevel < alarmLevel |
| R-OP-01 | 同型号水泵，w_pump 相同 |
| R-OP-03 | 备用泵同规格，数量 N_spare（可为零） |
| R-OP-05 | 水泵左右对称布置 |
| GB/T 12221 | 闸阀/止回阀面对面尺寸 |
| GB/T 12459 | 90° 弯头中心到端面 = 1.5×DN |
| GB 50265-2022 §7 | 管外壁到墙面净距 ≥ 800 mm |

## 七、输出

```json
{
  "L_room": "<维护间净长，m>",
  "W_room": "<维护间净宽，m>",
  "d_spacing": "<实际泵间净距，m>",
  "e_wall": "<端部距墙净距，m>",
  "svg": "<SVG 元素，嵌入网页展示>"
}
```

SVG 包含：
- 左侧平面图（俯视，设备 + 拓扑管道 + 管件实际比例符号）
- 右侧 A-A 剖面图（竖向空间 + 三水位标注 + 切割线可拖动 + 管道圆截面按位置变化）
- 工具栏（放大/缩小/重置）带最小缩放保护
