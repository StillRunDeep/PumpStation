# AG3-1 设备流线二维示意

> 功能分类：**图形表达**
> 本体对象定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG3-1 根据 AG2-1 计算的空间尺寸（已集成拓扑分析），将所有设备按实际比例摆入维护间，并生成两个视图：

- **平面图（俯视）**：设备水平布局、管件（弯头/止回阀/闸阀/流量计）按实际比例绘制
- **A-A 剖面图**：竖向空间关系，三水位标注，切割线可拖动

**简化颜色方案：**
- 泵：#2471a3
- 管道：#2980b9
- 阀门：#c0392b

---

## 二、在系统中的位置

```
AG0-0（工作泵台数 N）              → AG2-1
AG0-1（拓扑配置）                  → AG2-1（计算管道数量和阀门）
AG1-2（水泵选型）                  → P_motor, catalogPump
AG1-3（管道尺寸计算）              → DN_branch, DN_main
AG1-1（污水池计算）                → h_pool, Z_stop, Z_start, Z_alarm
AG2-1（维护间尺寸，含拓扑分析）    → L, W, DN_branch, DN_main, numPipes
  ▼
AG3-1（设备流线二维示意）
  │  输出：SVG 平面图 + A-A 剖面图
  ▼
网页展示区域
```

---

## 三、输入参数

> 注：系统采用潜污泵（Submersible Pump），泵浸没于集水坑内，无吸入管（PumpInlet）。

| 参数名 | 来源 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| N | AG0-0 | 台 | 工作泵台数 |
| ag21 | AG2-1 | — | 维护间尺寸计算结果 |
| params | AG1-1 | — | 水位参数 { h_active, Z_stop, Z_start1, Z_start2, Z_alarm_high, Z_alarm_low, Z_max } |
| S | AG1-1 | m² | 集水坑面积 |
| extraInfo | AG0-0/AG1-2 | — | { Q_single, H_design, P_motor, catalogPump, Z_sump } |

## 四、绘制内容

### 4.1 坐标系

| 视图 | X 轴 | Y 轴 | 说明 |
|------|------|------|------|
| 平面图 | 向右（泵排列方向L）| 向下（通道方向W）| 俯视 |
| A-A 剖面图 | 室内净宽 W | 向上（高程）| 竖向 |

### 4.2 平面图绘制对象

| 对象 | 绘制方式 | 颜色 |
|------|----------|------|
| 维护间外轮廓 | 矩形，L × W | #2980b9 |
| 集水坑区域 | 矩形，面积 S | #2980b9 |
| 水泵机组 | 矩形块 | #2471a3 |
| 压水管（竖向穿楼板）| 虚线圆圈 + 竖向虚线 | #2980b9 |
| 分支管路 | 单线 | #2980b9 |
| 止回阀、闸阀 | 标准符号 | #c0392b |
| MainOutlet 干管 | 横向粗线 | #2980b9 |
| DN 标注 | 单线旁标注 | #555 |

### 4.3 A-A 剖面图

| 要素 | 表达方式 | 说明 |
|------|----------|------|
| 维护间 | 矩形，W × h_room | 地上部分 |
| 水池 | 矩形，W × h_pool | 地下部分 |
| 水面示意 | 蓝色填充 | 从最高水位到池底 |
| 水位标注 | 水平虚线 + 标注 | 停泵/启泵/报警/最高水位 |
| 水泵机组（剖面）| 矩形 | 集水坑内 |
| 压水管 | 竖向线段 | 穿楼板 |
| 管道截面 | 圆形截面 | 按位置变化 |

**剖面切割线：**
- 沿L方向切割
- 可拖动调整位置
- 按位置显示不同截面：泵出口/中间管道区/总管+流量计

## 六、输出

SVG 包含：
- 左侧平面图（俯视，设备 + 拓扑管道 + 管件实际比例符号）
- 右侧 A-A 剖面图（竖向空间 + 水位标注 + 切割线可拖动 + 管道圆截面按位置变化）
- 工具栏（放大/缩小/重置）
