# AG2-1 泵房维护间尺寸计算

> 功能分类：**计算**
> 代码文件：`src/agents/maintenance-room.js`

---

## 一、职责概述

AG2-1 根据工作泵台数、备用泵台数和电机功率，计算泵房维护间的最小净尺寸（长度 L 和宽度 W），供 AG3-1 绘制二维平面图使用。

若上游 AG1-2 选型命中泵目录（含 `dimensions_mm`），则使用实际泵外形尺寸；否则使用通用默认值（0.6×0.8 m）。

---

## 二、在系统中的位置

```
AG0-0（工作泵台数 N、备用泵台数 N_spare）→ AG2-1
AG1-2（电机功率 P_motor）                → AG2-1（接收）
                                          ▼
                              AG2-1（维护间尺寸 L、W）
                                          │
                                          ▼
                              AG3-1（设备流线二维示意）
```

## 三、输入参数

| 参数名 | 来源 | 单位 | 说明 |
|--------|------|------|------|
| N | AG0-0 | 台 | 工作泵台数 |
| N_spare | AG0-0 | 台 | 备用泵台数，默认 0 |
| P_motor | AG1-2 | kW | 电机功率（参考用，废弃推算逻辑）|

| DN_main | AG1-3 | mm | 主管公称直径（总出水管），默认 300 |
| d_spacing | 用户/调用侧 | m | 泵间净距，直接输入 |
| e_wall | 用户/调用侧 | m | 端部距墙净距，直接输入 |
| spaceRules | 用户输入 | — | 可选，间距规则（覆盖默认值）|

## 四、计算逻辑

### 4.1 泵外形尺寸

| 来源 | w_pump（沿排列方向）| d_pump（沿通道方向）|
|------|---------------------|---------------------|
| catalogPump（含 dimensions_mm）| `dimensions_mm.b / 1000` m | `dimensions_mm.a / 1000` m |
| 默认值 | 0.6 m | 0.8 m |

### 4.2 间距规则（直接输入）

> **本项目遵循 DSD 渠务署香港规范。** GB 50265-2022 §7 功率→间距规则仅作技术参考，不作为硬编码默认值。

AG2-1 接受 `d_spacing` 和 `e_wall` 作为直接输入参数，由调用侧（AG3-1 / main.js）依据项目具体要求确定并传入。

默认参考值（仅作保底）：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| d_spacing | 1.0 m | 泵间净距 |
| e_wall | 0.8 m | 端部距墙净距 |

### 4.3 总布置台数

```
N_total = N + N_spare
```

### 4.4 维护间净长

```
L_raw = N_total × w_pump + (N_total - 1) × d_spacing + 2 × e_wall
L = ceilTo01(L_raw)    （向上取至 0.1 m 精度）
```

**示例：** N = 2，N_spare = 0，w_pump = 0.6 m，d_spacing = 1.0 m，e_wall = 0.8 m
→ N_total = 2，L = 2×0.6 + 1×1.0 + 2×0.8 = **4.0 m**

### 4.5 维护间净宽 — W_pipe 管件空间计算（新方法）

沿通道方向，从集水坑侧墙向远侧墙逐项累加：

```
c_wall   = (pipeToWall_mm + DN_branch/2) / 1000       // 管中心离集水坑墙 (m)
L_elbow  = elbowCTF(DN_branch) / 1000                  // 弯头水平投影 = 1.5×DN (m)
L_str    = max(minStraight_mm, 2×DN_branch) / 1000    // 阀件间直管段 (m)
L_cv     = lookupFF(CHECK_VALVE_FF, DN_branch) / 1000 // 止回阀面对面 (m)
L_gv     = lookupFF(GATE_VALVE_FF, DN_branch) / 1000   // 闸阀面对面 (m)

W_pipe = c_wall + L_elbow + L_str + L_cv + L_str + L_gv
       + (pipeToPipe_mm + DN_branch/2 + DN_main/2) / 1000  // 至总管中心
       + (pipeToWall_mm + DN_main/2) / 1000                // 总管到远侧墙
```

传统计算（保留作下限）：

```
W_equip = d_pump + 0.5    （水泵深度 + 出水管附件横向空间）
W_legacy = max(1.2, W_equip) + 0.3
```

最终取：

```
W = max(2.5, ceilTo01(max(W_pipe, W_legacy)))
```

取 W ≥ 2.5 m（工程最小值）。

### 4.6 W_breakdown（逐项明细）

| 项目 | 计算值 |
|------|--------|
| 管外壁到墙面净距 | pipeToWall_mm mm（GB 50265-2022 §7）|
| 弯头（1.5×DN）| elbowCTF(DN_branch) mm |
| 阀件间直管段 max(2D, minStraight) | max(2×DN_branch, minStraight_mm) mm |
| 止回阀面对面 | lookupFF(CHECK_VALVE_FF, DN_branch) mm（GB/T 12221）|
| 闸阀面对面 | lookupFF(GATE_VALVE_FF, DN_branch) mm（GB/T 12221）|
| 相邻管外壁间净距 | pipeToPipe_mm mm（GB 50265-2022 §7）|
| 总管外壁到远侧墙 | (pipeToWall_mm + DN_main) mm |

## 五、输出参数

| 参数名 | 单位 | 说明 |
|--------|------|------|
| L | m | 维护间净长（向上取 0.1 m） |
| W | m | 维护间净宽（≥ 2.5 m） |
| w_pump | m | 单泵外形宽度 |
| d_pump | m | 单泵外形深度 |

| e_wall | m | 端部距墙净距 |
| N_total | 台 | 总布置台数 |

| W_breakdown | array | W 各项明细 [{label, val, unit}] |
| sumpFittings | object | 集水坑管件说明 {elbow_vert_mm, reducer_mm, notes} |
| hasCatalogDims | bool | 是否使用 catalog 实际尺寸 |
| DN_branch, DN_main | mm | 当前管径 |
| c_wall_m, L_elbow_m | m | 管件坐标（供 AG3-1 绘图用）|

## 六、引用规则

| 规则 ID | 简述 | 来源 |
|---------|------|------|
| R-LA-01 | 相邻水泵净距 ≥ 1.0 m（大型泵 ≥ 1.2 m）| GB 50265-2022 §7 |
| R-LA-02 | 泵距侧墙 ≥ 0.8 m（大型泵 ≥ 1.0 m）| GB 50265-2022 §7 |
| GATE_VALVE_FF | 闸阀面对面尺寸 | GB/T 12221-2005 PN10 长型 |
| CHECK_VALVE_FF | 旋启式止回阀面对面尺寸 | GB/T 12221-2005 |
| elbowCTF | 90° 长半径弯头中心到端面 = 1.5×DN | GB/T 12459-2005 |
| reducerL | 偏心大小头长度 ≈ max(100, 1.5×\|D1-D2\|) | GB/T 12459-2005 |
| fmBodyL | 电磁流量计本体长度 ≈ max(300, 4×DN) | GB/T 18940 |
| pipeToWall_mm | 管外壁到墙面净距 ≥ 800 mm | GB 50265-2022 §7 |
| pipeToPipe_mm | 相邻管外壁间净距 ≥ 800 mm | GB 50265-2022 §7 |
