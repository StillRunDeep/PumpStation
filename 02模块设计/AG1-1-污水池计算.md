# AG1-1 调蓄池计算

> 功能分类：**数值计算**
> 依据：香港渠务署《雨水排水手册（第五版）》第14章
> 英文对象名：`WetWell`
> 代码文件：`src/agents/ag11-pool-depth.js`

---

## 一、职责概述

AG1-1 根据设计参数和水缸几何参数，计算：
1. **调蓄池容积校验**（步骤7）：验证设计容积是否满足最小调节容积要求
2. **控制水位**（停泵水位、启泵水位、最高水位、报警水位）（步骤6）

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  V_design, Z_bottom, D, N, Z, Q_pump
  ▼
AG1-1（调蓄池计算）
  │  输出：V_min, V_effective, Z_stop, Z_start1, Z_start2, Z_max, Z_alarm
  ▼
AG1-2（水泵选型，使用 Z_stop 计算扬程）
AG3-1（设备流线二维示意，使用三水位绘制剖面图）
```

## 三、输入参数

### 来自 AG0-0

| 参数名 | 符号 | 单位 | 说明 |
|--------|------|------|------|
| 设计水缸容量 | V_design | m³ | 用户输入的有效调蓄容积 |
| 池底标高 | Z_bottom | mPD | 集水池底部标高 |
| 设计水缸深度 | D | m | Z_top = Z_bottom + D；与 A_base 二选一 |
| 池底面积 | A_base | m² | 调蓄池底部面积；与 D 二选一 |
| 工作水泵台数 | N | 台 | - |
| 每小时允许启动次数 | Z | 次/小时 | 泵参数限制 |
| 单泵设计流量 | Q_pump | m³/s | Q_total / N |

### 几何模式规则

| 条件 | 模式 | 说明 |
|------|------|------|
| D 有值且 A_base 为空 | 模式A | 已知深度，推算面积 A_base = V_design / D |
| A_base 有值且 D 为空 | 模式B | 已知面积，推算深度 D = V_design / A_base |
| D 和 A_base 都有值 | 错误 | 只能提供其中一个 |
| D 和 A_base 都为空 | 错误 | 至少需要提供其中一个 |

### 设计参数（带默认值）

| 参数名 | 符号 | 单位 | 默认值 | 取值范围 | 依据 | 说明 |
|--------|------|------|-------|----------|------|------|
| 停泵淹没深度 | h_sub | m | 0.5 | 0.3-1.0 | 工程惯例≥0.5m | 防止泵吸入空气 |
| 1#泵启动水位系数 | k1 | - | 0.4375 | 0.40-0.50 | 手册第14.6.1节 | 池深比例（案例调整） |
| 2#泵启动水位系数 | k2 | - | 0.75 | 0.75-0.85 | 手册第14.6.1节 | 池深比例（案例调整） |
| 超高 | F_b | m | 1.3 | 0.5-2.0 | 设计参考 | 最高水位距池顶 |
| 安全余量 | F_s | m | 0.5 | 0.1-0.5 | 工程惯例 | 高于启泵水位的安全超高 |
| 低水位报警偏移 | h_alarm_offset | m | 0.5 | 0.3-0.5 | 工程惯例 | 防止报警抖动 |

### 水位关系校验规则

| 校验条件 | 依据 |
|---------|------|
| Z_stop ≥ Z_bottom | 停泵水位不得低于池底 |
| Z_start1 > Z_stop | 1#泵启动水位应高于停泵水位 |
| Z_start2 > Z_start1 | 2#泵启动水位应高于1#泵启动水位 |
| Z_max < Z_top - F_b | 最高水位应低于池顶减去超高 |
| Z_alarm_low ≥ Z_bottom | 低水位报警不得低于池底 |

## 五、计算逻辑

### 步骤 3：几何参数计算

#### 3.1 池顶标高

```
Z_top = Z_bottom + D    （mPD）
```

#### 3.2 集水坑底标高

```
Z_sump = Z_bottom - Z_sump_offset    （mPD）
```

#### 3.3 有效调蓄深度（至最高水位）

```
h_effective = Z_max - Z_stop    （m）
```

其中 Z_max 由步骤6确定。

---

### 步骤 7：调蓄演算与容积校验

**依据**：手册第14.5.2节、工程惯例值参考

#### 7.1 最小调节容积（由启动次数推算）

```
V_min = Q_pump / (4 × Z) × 3600    （m³）
```

其中：
- Q_pump = 单泵流量（m³/s）
- Z = 每小时允许启动次数（次/小时）
- 公式来源：工程惯例值参考 §2，当来水量等于泵流量一半时，启停最频繁，此时调节容积需求最大

#### 7.2 有效调蓄容积

```
V_effective = S × (Z_max - Z_stop)    （m³）
```

#### 7.3 容积校验

```
V_effective ≥ V_min    （校验通过）
V_effective ≥ V_design × k_utilization    （校验通过，k_utilization通常取0.8-0.9）
```

#### 7.4 每小时启动次数校核（步骤7输出）

依据规则 R-CM-02：
```
N_starts_actual ≤ Z    （泵参数限制）
```

---

### 步骤 6：控制水位设计

**依据**：手册第6.5节、第14.6.1节

#### 6.1 停泵水位（Pump Stop Level）

```
Z_stop = Z_bottom    （mPD）
```

主泵的停泵水位即为主水池池底标高。水位降至池底时泵停止，防止泵空抽。

**集水坑说明**：集水坑（Z_sump_offset）用于收集池底局部低洼处积水，由独立的集水坑泵排走，与主泵系统无关。

**限制条件**：
```
Z_stop ≥ Z_bottom
```

#### 6.2 1#泵启动水位（First Pump Start Level）

```
Z_start1 = Z_stop + k1 × D    （mPD）
```

其中 k1 = **0.4375**（案例调整值，原始手册范围0.40-0.50）

#### 6.3 2#泵启动水位（Second Pump Start Level）

```
Z_start2 = Z_stop + k2 × D    （mPD）
```

其中 k2 = **0.75**（案例调整值，原始手册范围0.75-0.85）

#### 6.4 最高水位（Maximum Water Level）

```
Z_max = Z_top - F_b - F_s    （mPD）
```

其中：
- F_b = 超高（默认1.3m）
- F_s = 安全余量（默认0.5m），高于启泵水位的安全超高

**限制条件**：
```
Z_max ≤ Z_top - 0.8 m    （超高验证）
```

#### 6.5 报警水位（Alarm Levels）

```
Z_alarm_low = Z_bottom - h_alarm_offset    （mPD，低水位报警）
Z_alarm_high = Z_start2    （mPD，高水位报警）
```

其中 h_alarm_offset = **0.5 m**（案例调整值，原0.3m）

> 注：
> - 低水位报警以池底为基准，而非集水坑底。
> - 高水位报警与2#泵启动水位一致，表示双泵均已启动水位仍上涨。

---

### 多泵梯级启动水位（N > 2 时）

依据规则 R-CL-01（多泵梯级启动规则）：

```
Pump_1.startLevel = Z_start1
Pump_2.startLevel = Z_start2
Pump_3.startLevel = Z_start2 + Δh    （Δh ≥ 0.2 m）
Pump_n.startLevel = Z_start(n-1) + Δh
所有泵统一停泵水位 = Z_stop
```

## 八、示例计算（真实案例复现）

**已知**：Z_bottom=-13.0 mPD, D=16 m, V_design=75000 m³, Q_total=3 m³/s, N=2, Z=8次/小时

**设计参数**（默认值为案例值）：
- k1 = 0.4375
- k2 = 0.75
- F_b = 1.3 m
- F_s = 0.5 m
- h_alarm_offset = 0.5 m

**步骤 3**：
```
Z_top = -13.0 + 16 = 3.0 mPD
```

**步骤 6**：
```
Z_stop = Z_bottom = -13.0 mPD  ← 主泵停泵水位
Z_start1 = -13.0 + 0.4375×16 = -6.0 mPD
Z_start2 = -13.0 + 0.75×16   = -1.0 mPD
Z_max = 3.0 - 1.3 - 0.5     = 1.2 mPD  ← 案例目标值
```

**步骤 7**：
```
Q_pump = 3/2 = 1.5 m³/s
V_min = 1.5/(4×8) × 3600 = 168.75 m³
```

**对比目标结果**：
| 参数 | 计算值 | 目标值 | 状态 |
|------|--------|--------|------|
| Z_stop | -13.0 mPD | -13.0 mPD | ✓ |
| Z_start1 | -6.0 mPD | -6.0 mPD | ✓ |
| Z_start2 | -1.0 mPD | -1.0 mPD | ✓ |
| Z_max | +1.2 mPD | +1.2 mPD | ✓ |
