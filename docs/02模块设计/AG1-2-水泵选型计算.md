# AG1-2 水泵选型

> 功能分类：**数值计算**
> 依据：香港渠务署《雨水排水手册（第五版）》第14章
> 本体对象与共享常量定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)
> 代码文件：`src/agents/ag12-pump-spec.js`

---

## 一、职责概述

AG1-2 根据设计流量、扬程和效率参数，计算水泵的：
1. **系统扬程**（步骤4）
2. **轴功率与电机功率**（步骤4）
3. **管道尺寸估算**
4. **流速校验与NPSH校验**

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  Q, Q_single, N, Z_bottom, Z_top

```

## 五、计算逻辑

### 步骤 4：水泵选型计算

**依据**：手册第14.6.2节

#### 4.1 单泵设计流量

```
Q_pump = Q_single / 3600    （m³/s）
```

#### 4.2 静扬程计算

```
H_static = Z_discharge - Z_stop    （m）
```

其中：
- Z_discharge：出水口标高（排放点）
- Z_stop：停泵水位（来自 AG1-1）

# 泵看到的压力差：
  吸入侧压力 = ρg × (Z_water - Z_suction)    ← 泵吸入口上方的水压
  排出侧压力 = ρg × (Z_discharge - Z_suction) + 管道损失

泵扬程 H = (排出 - 吸入) / ρg
         = (Z_discharge - Z_suction) - (Z_water - Z_suction) + 损失
         = Z_discharge - Z_water + 损失

# Z_suction（泵吸入口标高）在两边各出现一次，相互抵消！

所以：

H_static = Z_discharge - Z_water（当前水面标高）

H_static_max = Z_discharge - Z_stop = 7.0 - (-13.0) = 20.0 m   ← 用于泵选型（最大扬程）
H_static_min = Z_discharge - Z_max  = 7.0 - 1.2    = 5.8 m    ← 用于泵选型（最小扬程）

Z_alarm_low 有什么用？
Z_alarm_low 只在 **NPSH（汽蚀余量）**校验中有意义——如果报警触发时泵万一还在跑，要确认吸入口仍有足够淹没深度，不发生汽蚀。这是安全校验，不是主扬程计算。

#### 4.3 管道尺寸初估

**出水管计算内径**：
```
D_out = sqrt(4 × Q_pump / (π × v_out)) × 1000    （mm）
```

**进水管计算内径**：
```
D_in = sqrt(4 × Q_pump / (π × v_in)) × 1000    （mm）
```

#### 4.4 沿程损失（曼宁公式）

**依据**：手册第8.3节

```
H_f = 10.29 × n² × L × Q_pump² / D^5.33    （m）
```

其中：
- n：曼宁粗糙系数（混凝土=0.013）
- L：管长（m）
- D：管径（m）

#### 4.5 局部损失

```
H_local = k_local × H_f    （m）
```

其中 k_local = 0.15（通常为沿程损失的15%）

#### 4.6 总水头损失

```
H_loss = H_f + H_local    （m）
```

#### 4.7 系统总扬程

```
H_total = H_static + H_loss    （m）
```

---

### 轴功率计算

**依据**：手册第14.6.2节

#### 4.8 轴功率

```
ρ = 1000 kg/m³（水密度）
g = 9.81 m/s²（重力加速度）
η_combined = η_hyd × η_mot

P_shaft = ρ × g × Q_pump × H_total / (η_hyd × η_mot) / 1000    （kW）
```

#### 4.9 电机安全系数

依据总纲 §4.2：

| 轴功率范围 | 安全系数 K |
|-----------|-----------|
| P_shaft < 15 kW | 1.25 |
| 15 ≤ P_shaft < 55 kW | 1.15 |
| P_shaft ≥ 55 kW | 1.10 |

#### 4.10 电机功率

```
P_motor = P_shaft × K    （kW）
```

## 六、管道尺寸与流速校验

### 标准管径系列

DN 系列（mm）：100, 125, 150, 200, 250, 300, 350, 400, 450, 500, 600, 700, 800, 900, 1000, 1200, 1250, 1500

取值原则：取系列中**不小于计算内径的最小值**。

### 流速校验

**依据**：手册第8.3节

| 管道类型 | 允许范围 | 依据 |
|---------|---------|------|
| 进水管 | 0.6 ≤ v ≤ 1.2 m/s | 防止汽蚀 |
| 出水管 | 1.0 ≤ v ≤ 1.8 m/s | 经济流速 |

**实际流速计算**：
```
v_actual = Q_pump / (π × (D/2)²)    （m/s）
```

## 七、NPSH 校验

### 必需汽蚀余量

| 参数 | 符号 | 默认值 | 依据 |
|------|------|-------|------|
| 大泵 NPSH_r | NPSH_r | 3.0 m | 大型轴流泵典型值 |
| 小泵 NPSH_r | NPSH_r | 2.0 m | 中型离心泵典型值 |

### 有效汽蚀余量

**简化计算**：
```
NPSH_a = 10.33 - 0.5 + H_s - 0.2    （m）
```

其中：
- H_s = 淹没深度 = |Z_stop - Z_sump|

### 校验要求

```
NPSH_a ≥ NPSH_r + 0.5 m    （安全余量）
```

## 十、示例计算

**已知**：Q_single=936 m³/h, Z_stop=-12.0 mPD, Z_discharge=5.0 mPD, L=50m

**设计参数**：
- n = 0.013（手册第8.3节）
- η_hyd = 0.82（手册第14.6节）
- η_mot = 0.93（手册第14.6节）
- NPSH_r = 3.0 m（工程惯例）
- v_in = 1.0 m/s（手册第8.3节）
- v_out = 1.5 m/s（经济流速）
- k_local = 0.15（工程惯例）

**步骤 4**：
```
Q_pump = 936/3600 = 0.26 m³/s
H_static = 5.0 - (-12.0) = 17.0 m

D_out = sqrt(4×0.26/(π×1.5)) × 1000 = 470 mm → DN500
H_f = 10.29 × 0.013² × 50 × 0.26² / 0.5^5.33 = 0.87 m
H_local = 0.15 × 0.87 = 0.13 m
H_loss = 0.87 + 0.13 = 1.0 m
H_total = 17.0 + 1.0 = 18.0 m

P_shaft = 1000 × 9.81 × 0.26 × 18.0 / (0.82 × 0.93) / 1000 = 53.4 kW
P_shaft ≥ 55? 否 → K = 1.15
P_motor = 53.4 × 1.15 = 61.4 kW
```
