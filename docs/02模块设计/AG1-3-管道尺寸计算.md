> 功能分类：**数值计算**

## 一、职责概述

AG1-3 根据单泵流量和总流量，计算：
1. **管道公称内径**（DN）：吸水管、压水管、总出水管
2. **水力校核**：沿程损失、流速校验、NPSH 校验

## 五、计算逻辑

### 步骤 6a：管道尺寸计算

**依据**：手册第8.3节

#### 通用公式

将流量由 m³/h 转换为 m³/s，由连续性方程计算管径：

```
D_calc = sqrt(4 × Q_vol / (π × v)) × 1000    （mm）
```

再从 DN 标准系列（见总纲 §4.1）中取**不小于 D_calc 的最小值**，得公称内径 DN。

#### 6.1 PumpInlet 管径（吸水管，每台泵独立）

设计流速 v_in = **1.0 m/s**（防止流速过高引起汽蚀）：

```
D_pumpIn = sqrt(4 × Q_pump / (π × v_in)) × 1000    （mm）
DN_pumpIn = ceil_to_DN_series(D_pumpIn)
```

#### 6.2 PumpOutlet 管径（压水管，每台泵独立）

设计流速 v_out = **1.5 m/s**：

```
D_pumpOut = sqrt(4 × Q_pump / (π × v_out)) × 1000    （mm）
DN_pumpOut = ceil_to_DN_series(D_pumpOut)
```

#### 6.3 MainOutlet 管径（总出水干管）

总出水管承载所有工作泵合流后的总流量 Q：

```
q_total = Q / 3600    （m³/s）
D_mainOutlet = sqrt(4 × q_total / (π × v_out)) × 1000    （mm）
DN_mainOutlet = ceil_to_DN_series(D_mainOutlet)
```

---

### 步骤 6b：水力校核

**依据**：手册第8.3节、第14.2.3节

#### 6.4 沿程损失校验（曼宁公式）

```
H_f = 10.29 × n² × L × Q_pump² / D^5.33    （m）
```

其中：
- n：曼宁粗糙系数
- L：管长（m）
- D：管径（m）

#### 6.5 局部损失

```
H_local = k_local × H_f    （m）
```

#### 6.6 总水头损失

```
H_loss = H_f + H_local    （m）
```

#### 6.7 流速校验

```
v_actual = Q_pump / (π × (D/2)²)    （m/s）
```

**校验标准**：

| 管道类型 | 允许范围 | 依据 |
|---------|---------|------|
| 进水管 | 0.6 ≤ v ≤ 1.2 m/s | 防止汽蚀 |
| 出水管 | 1.0 ≤ v ≤ 1.8 m/s | 经济流速 |

#### 6.8 NPSH 校验

**必需汽蚀余量**（设备参数，由泵厂提供）：
```
NPSH_r = 3.0 m    （大型轴流泵典型值）
```

**有效汽蚀余量**：
```
NPSH_a = (P_atm - P_v) / (ρg) + H_s - H_suction_loss    （m）
```

简化计算：
```
NPSH_a = 10.33 - 0.5 + H_s - 0.2    （m）
```

其中：
- P_atm = 101.325 kPa（标准大气压）
- P_v = 2.34 kPa（20°C时蒸汽压）
- H_s = 淹没深度 = |Z_stop - Z_sump|
- H_suction_loss = 进口损失 ≈ 0.2 m

**校验要求**：
```
NPSH_a ≥ NPSH_r + 0.5 m    （安全余量）
```

## 八、示例计算

**已知**：Q_pump=0.26 m³/s, Q=1872 m³/h, N=2, L=50m, NPSH_r=3.0m, Z_stop=-12.0 mPD, Z_sump=-12.5 mPD

**管道尺寸**：
```
D_pumpIn = sqrt(4×0.26/(π×1.0))×1000 = 575 mm → DN600
D_pumpOut = sqrt(4×0.26/(π×1.5))×1000 = 470 mm → DN500
q_total = 1872/3600 = 0.52 m³/s
D_mainOutlet = sqrt(4×0.52/(π×1.5))×1000 = 665 mm → DN700
```

**水力校验**：
```
H_f = 10.29 × 0.013² × 50 × 0.26² / 0.5^5.33 = 0.87 m
H_local = 0.15 × 0.87 = 0.13 m
H_loss = 1.0 m

v_in_actual = 0.26/(π×0.3²) = 0.92 m/s ✓
v_out_actual = 0.26/(π×0.25²) = 1.33 m/s ✓

H_s = |Z_stop - Z_sump| = |-12.0 - (-12.5)| = 0.5 m
NPSH_a = 10.33 - 0.5 + 0.5 - 0.2 = 10.13 m
NPSH_a ≥ 3.0 + 0.5 → 10.13 ≥ 3.5 ✓
```
