# AG2-1 泵房维护间尺寸计算

> 功能分类：**计算**
> 代码文件：`src/agents/maintenance-room.js`

---

## 一、职责概述

AG2-1 根据工作泵台数、备用泵台数、拓扑配置和电机功率，计算泵房维护间的最小净尺寸（长度 L 和宽度 W），供 AG3-1 绘制二维平面图使用。

**核心特点：**
- 从拓扑中分析管道数量和阀门配置
- L/W 计算基于拓扑中实际管道路径和阀门类型
- 阀门尺寸从 fitting-dims.js 查表获取（按实际阀门类型匹配）

若上游 AG1-2 选型命中泵目录（含 `dimensions_mm`），则使用实际泵外形尺寸；否则使用通用默认值（0.6×0.8 m）。

---

## 二、在系统中的位置

```
AG0-0（工作泵台数 N、备用泵台数 N_spare）→ AG2-1
AG0-1（拓扑配置：管道数量、阀门类型）     → AG2-1
AG1-2（电机功率 P_motor）                → AG2-1（接收）
AG1-3（DN_branch、DN_main）             → AG2-1
                                          ▼
                              AG2-1（维护间尺寸 L、W）
                                          │
                                          ▼
                              AG3-1（设备流线二维示意）
```

## 四、拓扑分析

### 4.1 从拓扑中提取关键信息

1. **找到汇合点 (junction)**：在 pump_room 中的 junction 设备
2. **追踪从 junction 出发的所有路径**：使用 DFS 追踪到房间边界
3. **计算管道数量**：`numPipes = 路径数量`
4. **识别最长路径的阀门**：按阀门类型查表获取尺寸

### 4.2 阀门尺寸查表

根据拓扑中阀门实际类型匹配：

| 阀门类型 | 尺寸查表 | 标准 |
|---------|---------|------|
| gate_valve | lookupFF(GATE_VALVE_FF, DN) | GB/T 12221 |
| check_valve | lookupFF(CHECK_VALVE_FF, DN) | GB/T 12221 |

---

## 五、计算逻辑

### 5.1 泵外形尺寸

| 来源 | w_pump（沿排列方向）| d_pump（沿通道方向）|
|------|---------------------|---------------------|
| catalogPump（含 dimensions_mm）| `dimensions_mm.b / 1000` m | `dimensions_mm.a / 1000` m |
| 默认值 | 0.6 m | 0.8 m |

### 5.2 间距规则（直接输入）

> **本项目遵循 DSD 渠务署香港规范。** GB 50265-2022 §7 功率→间距规则仅作技术参考，不作为硬编码默认值。

默认参考值（仅作保底）：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| d_spacing | 1.0 m | 泵间净距 |
| e_wall | 0.8 m | 端部距墙净距 |

### 5.3 总布置台数

```
N_total = N + N_spare
```

### 5.4 维护间净长 — L_pipe 管道占长计算

**公式：**
```
L = 墙距管道穿楼板洞的距离 + 弯头距离 + 最小直管段 + 阀门（按实际数量和类型）
    + 三通长度（汇合到总管的，用 DN_main 计算）+ 管道距墙的距离
```

**沿泵排列方向（L方向)，管道从汇合点到房间边界：**

```
// 基础管件尺寸
L_str   = max(minStraight_mm, 2×DN_branch) / 1000    // 阀件间直管段 (m)
L_gv    = lookupFF(GATE_VALVE_FF, DN_branch) / 1000   // 闸阀面对面 (m)
L_cv    = lookupFF(CHECK_VALVE_FF, DN_branch) / 1000 // 止回阀面对面 (m)
L_tee   = DN_main / 1000                              // 三通用 DN_main 尺寸 (m)
L_fm    = flowmeterBodyL(DN_main) / 1000              // 流量计长度 (m)

// 汇合点后的阀门总长度（按拓扑中实际阀门类型查表）
junction_length = Σ(各阀门尺寸) + L_tee + L_fm

// L_pipe 计算
L_pipe = pipeToWall + DN_branch + (pipeToPipe + DN_branch) × (numPipes - 1)
         + max(junction_length, pipeToWall)
```

**L 最终取：**
```
L_pumpBased = N_total × w_pump + (N_total - 1) × d_spacing + 2 × e_wall
L = ceilTo01(max(L_pipe, L_pumpBased))    （取管道计算值和泵排列计算值中的较大者）
```

### 5.5 维护间净宽 — W_pipe 管件空间计算

**沿通道方向（W方向)，从集水坑侧墙向远侧墙：**

```
W = 墙距管道穿楼板洞的距离 + 弯头距离 + 最小直管段 + 止回阀 + 最小直管段 + 闸阀 + 三通长度 + 管道距墙的距离

W_pipe = pipeToWall + L_elbow + L_str + L_cv + L_str + L_gv + L_tee + pipeToWall
```

**传统计算（保留作下限）：**
```
W_equip = d_pump + 0.5    （水泵深度 + 出水管附件横向空间）
W_legacy = max(1.2, W_equip) + 0.3
```

**W 最终取：**
```
W = max(2.5, ceilTo01(max(W_pipe, W_legacy)))    （取 W ≥ 2.5 m）
```

## 六、输出参数

| 参数名 | 单位 | 说明 |
|--------|------|------|
| L | m | 维护间净长（向上取 0.1 m） |
| W | m | 维护间净宽（≥ 2.5 m） |
| w_pump | m | 单泵外形宽度 |
| d_pump | m | 单泵外形深度 |
| d_spacing | m | 泵间净距 |
| e_wall | m | 端部距墙净距 |
| N_total | 台 | 总布置台数 |
| numPipes | 条 | 从汇合点到房间边界的管道数量 |

| W_breakdown | array | W 各项明细 [{label, val, unit}] |
| sumpFittings | object | 集水坑管件说明 {elbow_vert_mm, reducer_mm, notes} |

| DN_branch, DN_main | mm | 当前管径 |
| c_wall_m, L_elbow_m | m | 管件坐标（供 AG3-1 绘图用）|

## 七、引用规则

| 规则 ID | 简述 | 来源 |
|---------|------|------|
| R-LA-01 | 相邻水泵净距 ≥ 1.0 m（大型泵 ≥ 1.2 m）| GB 50265-2022 §7 |
| R-LA-02 | 泵距侧墙 ≥ 0.8 m（大型泵 ≥ 1.0 m）| GB 50265-2022 §7 |
| GATE_VALVE_FF | 闸阀面对面尺寸 | GB/T 12221-2005 PN10 长型 |
| CHECK_VALVE_FF | 旋启式止回阀面对面尺寸 | GB/T 12221-2005 |
| elbowCTF | 90° 长半径弯头中心到端面 = 1.5×DN | GB/T 12459-2005 |
| flowmeterBodyL | 电磁流量计本体长度 ≈ max(300, 4×DN) | GB/T 18940 |
| pipeToWall_mm | 管外壁到墙面净距 ≥ 800 mm | GB 50265-2022 §7 |
| pipeToPipe_mm | 相邻管外壁间净距 ≥ 800 mm | GB 50265-2022 §7 |
