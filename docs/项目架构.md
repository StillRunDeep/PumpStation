# PumpStation 项目架构总结

## 一、项目概述

**PumpStation** 是一个**雨水泵站智能设计系统**，基于浏览器（Vite + ES Module）实现的雨水泵房设计 AI 工具。

- **项目名称**: pump-station
- **版本**: 0.1.0
- **技术栈**: Vite (5.4.0) + ES Module
- **主要功能**: 雨水泵站设计计算、布局生成、方案评估
- **部署地址**: https://pumpstation.ethanmiao.cn/

## 四、核心模块说明

### 1. AG0-0 用户参数解析器 (src/agents/user-params.js)

**职责**:
- 解析用户输入的基本参数（排水量、池深、池底面积、泵台数等）
- 校验参数合法性
- 确定计算模式（直接模式 / 暴雨模式）
- 提供拓扑基础信息（N, N_spare 等）

**关键参数**:
- `Q_total` 总排水量 (m³/s)
- `V_design` 设计水缸容量 (m³)
- `D` 池深 (m)
- `N` 工作泵台数
- `N_spare` 备用泵台数

---

### 2. AG0-1 暴雨计算模块 (src/agents/topology.js)

**职责**:
- 暴雨强度查询 (IDF 公式)
- 径流系数计算
- 设计降雨量计算
- 拓扑连接关系验证

**数据来源**:
- 香港天文台总部 IDF 曲线
- 大帽山、西部大屿山等不同降雨分区数据
- 重现期 T = 10 年 / 50 年 / 200 年

---

### 3. AG1-1 污水池计算模块 (src/agents/pool-depth.js)

**职责**:
- 调蓄池容积计算
- 启动频率约束校验
- 集水池深度与面积反推
- 止水标高的确定

**核心公式**:
- 启动次数约束：`Z = (3600 × Q_pump) / (V_design - V_occupied)`
- 池深 `D = (Z_max - Z_stop) / Z_alarm_margin`

---

### 4. AG1-2 水泵选型模块 (src/agents/pump-spec.js)

**职责**:
- 水泵扬程计算 (静扬程 + 动扬程 + 管损)
- 电机功率估算
- 模型匹配 (从 product catalog)
- NPSH 汽蚀余量校验

**计算内容**:
- `H_total = H_static + H_velocity + H_loss`
- `P_motor = (ρ × g × Q × H_total) / (η_hyd × η_mot × 1000)`

---

### 5. AG1-3 管道尺寸计算模块 (src/agents/pipe-sizing.js)

**职责**:
- 管径计算 (曼宁公式)
- 流速控制 (进水管 0.6-1.5 m/s, 出水管 1.5-3.0 m/s)
- 方案生成 (经济型/稳健节能型/高性能型)

**三套方案**:
- **方案 1 (经济型)**: 较小管径，较低流速
- **方案 2 (稳健节能)**: 平衡流速，综合成本低
- **方案 3 (高性能)**: 较大管径，较高流速

---

### 6. AG2-1 维护间尺寸计算模块 (src/agents/maintenance-room.js)

**职责**:
- 泵房维护间长宽计算
- 设备流线分析
- 管径与阀门配置
- 检修空间预留

**约束条件**:
- 泵间净距规范
- 端部距墙净距
- 管外壁到墙面净距
- 阀门间最小直管段

---

### 7. AG3-1 设备流线绘图模块 (src/agents/drawing.js + svg.js)

**职责**:
- 使用 SVG.js 绘制设备布置图
- 水位标注
- 警示水位线
- 设备名称与型号标注

---

### 8. AG4-1/2 布局生成与评价模块 (src/layout/)

**职责**:
- 建筑平面布局自动生成
- 多方案生成与评分
- Revit 格式导出

**算法**:
- **约束生长算法**: 基于空间约束的递归生长
- **评分器**: 多维评价指标 (合规性、经济性、可扩展性)
- **优化算法**: 边界条件优化

## 五、关键特性

### 1. Agent 架构模式

采用**Agent 编排模式**，每个功能模块独立封装为可复用组件：
- 每个 Agent 有独立的输入输出接口
- 支持模块化重算（上游变化触发下游重算）
- 通过 `moduleCache` 缓存中间结果

### 2. 计算链式依赖

```
AG0-0 → AG0-1 → AG1-1 → AG1-2 → AG1-3 → AG2-1 → AG3-1 → AG4-1
  (参数)  (暴雨)  (水池) (水泵)  (管道)  (维护间)  (绘图)  (布局)
```

### 3. 数据驱动设计

- 设备数据外部化 (`pump-catalog.js`)
- 评分规则参数化 (`scorer-params.js`)
- 空间约束文档化 (`room-defs.js`)

### 4. 可视化渲染

- 使用 SVG.js 进行矢量图形渲染
- 支持缩放、平移交互
- Tooltip 悬浮提示

### 5. 权限与安全

- Token 验证机制 (`src/auth.js`)
- 受保护的业务计算
