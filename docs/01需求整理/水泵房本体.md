# 雨水泵房本体定义

本文档定义雨水泵房领域的本体（Ontology），涵盖常用设备对象（Object）、对象间关系（Relation）以及工程约束规则（Rule），用于支持智能设计、自动校验和知识推理。

---

## 一、Object（对象/类）

### 1. RainfallPumpStation（雨水泵房）

顶层容器对象，代表整座雨水泵房设施。

| 属性名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | String | 唯一标识符 | 必填 |
| name | String | 泵房名称 | 必填 |
| layoutType | Enum | 布置形式：单排/双排对置/地下湿式 | 必填 |
| designFloodLevel | Float | 设计洪水位（m） | 必填 |
| designFlowRate | Float | 总设计流量 Q（m³/h） | > 0 |
| catchmentArea | Float | 汇水面积 A（m²） | > 0 |
| runoffCoefficient | Float | 径流系数 ψ | 0 < ψ ≤ 1 |
| returnPeriod | Int | 设计重现期（年） | ≥ 1 |

---

### 2. Pump（水泵）

雨水泵房核心排水设备，通常为离心泵。

| 属性名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | String | 唯一标识符 | 必填 |
| model | String | 型号 | 必填 |
| flowRate | Float | 单泵流量 Q（m³/h） | > 0 |
| head | Float | 扬程 H（m） | > 0 |
| efficiency | Float | 额定效率 η（%） | 0 < η ≤ 100 |
| npshRequired | Float | 必需汽蚀余量 NPSHr（m） | > 0 |
| installElevation | Float | 安装高程 Zs（m） | 见Rule R-HY-01 |
| isStandby | Bool | 是否为备用泵 | 默认 false |
| startupCount | Int | 每小时最大启动次数（次/h） | 4 ～ 6 |

---

| 属性名 | 类型 | 说明 | 约束 |
|--------|------|------|------|

| ratedPower | Float | 额定功率 P（kW） | > 0 |

### 4. CollectionPool（集水池）

收集来水并为水泵提供稳定进水条件的调节构筑物。

|--------|------|------|------|

| startLevel | Float | 启泵水位（m） | > stopLevel |
| stopLevel | Float | 停泵水位（m） | 见Rule R-CM-03 |
| alarmLevel | Float | 报警水位（m） | > startLevel |
| bottomSlope | Float | 底部坡度 | ≥ 1:10（0.1） |
| inletVelocity | Float | 进水流速（m/s） | 0.3 ～ 0.6 |

### 13. PumpCharacteristicCurve（水泵特性曲线）

描述单台水泵在不同流量下扬程、效率的 H-Q 曲线。

|--------|------|------|------|
| id | String | 唯一标识符 | 必填 |
| dataPoints | List\<{Q, H, η}\> | 曲线数据点列表 | 至少3个点 |
| bestEfficiencyPoint | {Q, H, η} | 最高效率工作点（BEP） | 必填 |
| minEfficiencyRatio | Float | 允许运行效率下限（相对BEP） | ≥ 0.85 |

---

### 14. SystemResistanceCurve（系统阻力曲线）

描述管路系统阻力随流量变化的曲线，H = H_静 + k·Q²。

| 属性名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | String | 唯一标识符 | 必填 |
| staticHead | Float | 静扬程 H_静（m） | ≥ 0 |
| resistanceCoefficient | Float | 阻力系数 k | > 0 |

## 二、Relation（关系）

| 关系名 | 主体（Domain） | 客体（Range） | 说明 |
|--------|----------------|---------------|------|
| **contains** | RainfallPumpStation | Pump \| CollectionPool \| PowerDistributionCabinet \| ControlCabinet \| HoistingEquipment | 泵房包含的设备组成 |
| **drives** | Motor | Pump | 电机驱动水泵 |
| **controls** | ControlCabinet | Motor | 控制柜控制电机启停 |
| **suppliesTo** | CollectionPool | Pump | 集水池向水泵提供来水 |
| **hasInletPipe** | Pump | InletPipe | 水泵连接独立进水管 |
| **hasOutletPipe** | Pump | OutletPipe | 水泵连接出水管 |
| **protectedBy** | OutletPipe | CheckValve | 出水管由止回阀保护防倒流 |
| **isolatedBy** | OutletPipe | GateValve | 出水管由闸阀进行隔断 |
| **parallelWith** | Pump | Pump | 两台水泵并联运行（对称关系） |
| **triggeredBy** | Pump | LevelSensor | 水泵启停由液位传感器信号触发 |
| **hasCharacteristicCurve** | Pump | PumpCharacteristicCurve | 水泵关联其H-Q特性曲线 |
| **hasSystemCurve** | RainfallPumpStation | SystemResistanceCurve | 泵房关联管路系统阻力曲线 |
| **monitoredBy** | CollectionPool | LevelSensor | 集水池由液位传感器监测 |
| **hasMotor** | Pump | Motor | 水泵配套电机 |

## 三、Rule（规则）

### 3.1 布置规则（Layout Rules）

---

**R-LA-01：水泵间距规则**

- **描述**：相邻水泵之间需留足维修通道
- **约束**：
  ```
  distance(Pump_i, Pump_j) ≥ 1.0 m        （相邻水泵净距，一般情况）
  distance(Pump_i, Pump_j) ≥ 1.2 m        （大型水泵）
  ```
- **触发条件**：任意两台相邻 Pump 对象
- **来源**：排水泵排布规则 §1

---

**R-LA-02：泵与墙壁间距规则**

- **描述**：水泵距墙壁需留足操作和检修空间
- **约束**：
  ```
  distance(Pump, Wall) ≥ 0.8 m             （一般情况）
  distance(Pump, Wall) ≥ 1.0 m             （大型水泵）
  ```
- **触发条件**：Pump 对象与墙壁边界
- **来源**：排水泵排布规则 §1

---

**R-LA-03：大型泵加大间距规则**

- **描述**：功率超过 55kW 的水泵须适当加大间距，便于吊装检修
- **约束**：
  ```
  IF Pump.motor.ratedPower > 55 kW
  THEN distance(Pump_i, Pump_j) > 1.2 m
  AND  distance(Pump, Wall) > 1.0 m
  ```
- **来源**：排水泵排布规则 §1

---

**R-LA-04：进水管独立取水规则**

- **描述**：各泵进水管必须独立，避免相互干扰抢水
- **约束**：
  ```
  FOREACH Pump p: p.inletPipe.isIndependent = true
  ```
- **来源**：排水泵排布规则 §1，集水池容量与水泵匹配规则 §3

---

**R-LA-05：进水流速规则**

- **描述**：进水流速须控制在合理范围，避免产生旋涡
- **约束**：
  ```
  0.3 m/s ≤ CollectionPool.inletVelocity ≤ 0.6 m/s
  ```
- **来源**：排水泵排布规则 §3

---

**R-LA-06：吸水口距底距离规则**

- **描述**：水泵吸水口与集水池底部和侧壁的最小距离
- **约束**：
  ```
  InletPipe.distanceToBottom ≥ 0.5 × InletPipe.diameter
  InletPipe.distanceToWall   ≥ 0.75 × InletPipe.diameter
  ```
- **来源**：排水泵排布规则 §3

---

**R-LA-07：电气设备安装高程规则**

- **描述**：配电柜和控制柜必须安装在设计洪水位以上的干燥区域
- **约束**：
  ```
  PowerDistributionCabinet.installElevation > RainfallPumpStation.designFloodLevel
  ControlCabinet.installElevation          > RainfallPumpStation.designFloodLevel
  ```
- **别名**：R-EL-01
- **来源**：排水泵排布规则 §5

---

**R-LA-08：就地控制面板高度规则**

- **描述**：就地控制箱面板中心距地高度须便于站立操作
- **约束**：
  ```
  1.4 m ≤ ControlCabinet.panelCenterHeight ≤ 1.5 m
  ```
- **来源**：排水泵排布规则 §5

---

### 3.2 水力规则（Hydraulic Rules）

---

**R-HY-01：水泵安装高程（汽蚀）规则**

- **描述**：水泵安装高程不得超过允许吸上高度，防止汽蚀损坏叶轮
- **约束**：
  ```
  Pump.installElevation ≤ NPSH_available - Pump.npshRequired - 0.5
  ```
  其中 NPSH_available 由当地大气压、水温、进水池水位计算得出
- **来源**：排水泵排布规则 §2

---

**R-HY-02：系统总扬程计算规则**

- **描述**：水泵设计扬程须覆盖系统总阻力，并含安全余量
- **约束**：
  ```
  Pump.head ≥ H_static + H_friction + H_local
  H_design  = (H_static + H_friction + H_local) × (1.05 ～ 1.10)
  ```
  - H_static：静扬程（进水最低水位至出水排放点高差）
  - H_friction：沿程摩擦损失（达西-韦斯巴赫公式或海曾-威廉公式）
  - H_local：局部阻力损失（弯头、阀门等）
- **来源**：洪水泵站水泵规格计算 §2

---

**R-HY-03：水泵工作点效率规则**

- **描述**：工作点（特性曲线与系统阻力曲线交点）必须处于水泵高效区
- **约束**：
  ```
  η_operating ≥ 0.85 × PumpCharacteristicCurve.bestEfficiencyPoint.η
  ```
- **来源**：洪水泵站水泵规格计算 §4

---

**R-HY-04：电机功率计算规则**

- **描述**：电机额定功率须满足水泵轴功率要求
- **公式**：
  ```
  P = (ρ × g × Q × H) / (η_pump × η_motor × η_trans)
  ```
  - ρ = 1000 kg/m³（水的密度）
  - g = 9.81 m/s²
  - Q：流量（m³/s）
  - H：扬程（m）
- **约束**：`Motor.ratedPower ≥ P × 安全系数（通常 1.1 ～ 1.2）`
- **来源**：洪水泵站水泵规格计算 §5

---

**R-HY-05：设计流量计算规则**

- **描述**：泵站总设计流量由汇水面积法或水文频率分析确定
- **公式**（汇水面积法）：
  ```
  Q = ψ × i × A
  ```
  - ψ：径流系数
  - i：设计降雨强度（m³/(s·m²) 或换算单位）
  - A：汇水面积（m²）
- **来源**：洪水泵站水泵规格计算 §1

---

### 3.3 容量匹配规则（Capacity Matching Rules）

---

**R-CM-01：集水池有效容积规则**

- **描述**：集水池有效调节容积须满足水泵启停频率要求
- **公式**：
  ```
  V ≥ Q_p / (4 × Z)
  ```
  - Q_p：单泵流量（m³/h）
  - Z：每小时最大启动次数（次/h），通常 4 ～ 6
- **约束**：`CollectionPool.effectiveVolume ≥ Q_p / (4 × Z)`
- **说明**：在来水量等于泵流量一半时，启停最频繁，此时调节容积需求最大
- **来源**：集水池容量与水泵匹配规则 §1

---

**R-CM-02：水泵启动频率规则**

- **描述**：水泵每小时启动次数不得超过电机允许上限
- **约束**：
  ```
  actualStartsPerHour ≤ Motor.maxStartsPerHour
  Motor.maxStartsPerHour ∈ [4, 6]          （离心泵典型值）
  ```
- **来源**：集水池容量与水泵匹配规则 §1

---

**R-CM-03：停泵水位淹没深度规则**

- **描述**：停泵水位必须高于水泵进口足够距离，防止吸入空气
- **约束**：
  ```
  (CollectionPool.stopLevel - Pump.inletElevation) ≥ 0.5 m
  ```
- **来源**：集水池容量与水泵匹配规则 §4

---

**R-CM-04：水位逻辑顺序规则**

- **描述**：集水池三水位设定值须满足逻辑顺序
- **约束**：
  ```
  stopLevel < startLevel < alarmLevel
  ```
- **来源**：集水池容量与水泵匹配规则 §2

---

**R-CM-05：集水池底部坡度规则**

- **描述**：集水池底部须有足够坡度，便于排除沉积物
- **约束**：
  ```
  CollectionPool.bottomSlope ≥ 0.1（即 1:10）
  ```
- **来源**：集水池容量与水泵匹配规则 §4

---

### 3.4 控制逻辑规则（Control Logic Rules）

---

**R-CL-01：梯级启泵规则**

- **描述**：多泵并联时，随水位升高依次投入更多水泵（梯级启动）
- **逻辑**：
  ```
  IF waterLevel ≥ Pump_1.startLevel  THEN start(Pump_1)
  IF waterLevel ≥ Pump_2.startLevel  THEN start(Pump_2)   // startLevel_2 > startLevel_1
  IF waterLevel ≥ Pump_3.startLevel  THEN start(Pump_3)   // startLevel_3 > startLevel_2
  ...
  IF waterLevel ≤ stopLevel          THEN stop(ALL running pumps)
  ```
- **来源**：集水池容量与水泵匹配规则 §3

---

**R-CL-02：报警水位响应规则**

- **描述**：水位超过报警水位时，触发告警并通知运维人员
- **逻辑**：
  ```
  IF waterLevel ≥ CollectionPool.alarmLevel
  THEN triggerAlarm() AND notify(operators)
  ```
- **来源**：集水池容量与水泵匹配规则 §2

---

**R-CL-03：独立格间进水规则**

- **描述**：每台水泵应有独立的进水格间，用隔墙分开
- **约束**：
  ```
  FOREACH Pump p: EXISTS IndependentIntakeChamber(c) WHERE c.servesPump = p
  ```
- **来源**：排水泵排布规则 §3

---

### 3.5 运行配置规则（Operational Configuration Rules）

---

**R-OP-01：同型号水泵规则**

- **描述**：同一泵站内水泵应尽量选用相同型号，便于备件管理和互换
- **约束**：
  ```
  COUNT(DISTINCT Pump.model IN RainfallPumpStation) = 1   （推荐）
  ```
- **约束强度**：建议（Recommended）
- **来源**：排水泵排布规则 §4

---

**R-OP-02：并联泵台数限制规则**

- **描述**：并联水泵台数不宜过多，否则单泵工作点偏移，效率大幅下降
- **约束**：
  ```
  COUNT(Pump WHERE isStandby = false IN RainfallPumpStation) ≤ 6
  ```
- **推荐范围**：4 ～ 6 台（不含备用）
- **来源**：排水泵排布规则 §4

---

**R-OP-03：备用泵配置规则（N+1）**

- **描述**：大型泵站应设置备用泵，确保运行可靠性
- **约束**：
  ```
  COUNT(Pump WHERE isStandby = true) ≥ 1
  ```
- **约束强度**：大型泵站（设计流量较大时）为强制，其余为建议
- **来源**：洪水泵站水泵规格计算 §6

---

**R-OP-04：备用电源配置规则**

- **描述**：洪水泵站需考虑洪水期电网供电可靠性，必要时配备备用电源
- **约束**：
  ```
  IF RainfallPumpStation.returnPeriod ≥ 20 年
  THEN PowerDistributionCabinet.hasBackupPower = true   （建议）
  ```
- **来源**：洪水泵站水泵规格计算 §6

---

**R-OP-05：布置对称性规则**

- **描述**：泵组设备布置应对称，使各泵进水条件尽量一致
- **约束**：
  ```
  FOREACH Pump p1, p2 IN PumpGroup:
    symmetricLayout(p1, p2) = true   （位置和进水管长度对称）
  ```
- **约束强度**：建议（Recommended）
