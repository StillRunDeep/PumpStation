## 五、管道系统布局（Piping Sequence）

参考图纸：4110365/BIN/PID/10001

```
水泵
 └─ DN1000 上升主管（Rising Main）
     └─ 组合式排气阀（CAV）
         └─ 止回阀（Check Valve）
             └─ 电动闸阀（Motorized Gate Valve）
                 └─ 汇合：2 × DN1000 + 1 回流管 → DN1200 上升主管
                     └─ 电磁流量计（Electromagnetic Flowmeter）
                         └─ 电动阀门（Motorized Gate Valve）
                             └─ 组合式排气阀（CAV）
                                 └─ 组合式排气阀（CAV）
                                     └─ DN1200 上升主管
```

**电磁流量计直管段要求（D = 1200 mm）：**

| 方向 | 要求 | 换算长度 |
|------|------|---------|
| 上游 | ≥ 5D | ≥ 6000 mm |
| 下游 | ≥ 3D | ≥ 3600 mm |
